# Algorithms – Seminar 3

Detta projekt innehåller tre separata uppgifter som tillsammans behandlar tre klassiska datastrukturer: heap, hashtabell och ordpusselösare. Varje uppgift har ett eget GUI för visualisering.

---

## Uppgifter

### Task 1 – Binär Heap
Implementering och analys av en min-heap med två konstruktionsmetoder (insert ett-i-taget respektive buildHeap), trädtraversaler och tidskomplexitetsmätningar.
→ Se [task1/README.md](task1/README.md) för detaljerad beskrivning.

### Task 2 – Hashtabell med kollisionshantering
Implementering av en hashtabell med tre olika strategier för kollisionshantering: separat länkning, linjär sondering och kvadratisk sondering.
→ Se [task2/README.md](task2/README.md) för detaljerad beskrivning.

### Task 3 – Ordpusselösare
En algoritm som söker igenom ett 2D-rutnät i alla 8 riktningar och hittar giltiga ord från en ordlista. Använder prefix-baserad beskärning för effektivitet.
→ Se [task3/README.md](task3/README.md) för detaljerad beskrivning.

---

## Köra projektet

Varje task har en egen `Main.java` som startar ett grafiskt gränssnitt (GUI). Kompilera och kör respektive `Main`-klass för att starta önskad uppgift.
